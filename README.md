Game Description:
An Empty Robot’s Heart is a story-driven exploration game set in a post-apocalyptic simulation. Play as AURA, an android trapped in a world of forgotten memories. Solve puzzles, discover hidden truths, and confront your purpose as you unravel the mystery of humanity’s past and your own existence.
